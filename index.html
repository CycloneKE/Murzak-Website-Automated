◘ MATATUS KE App
(Available on Google Playstore)



■ MATATUS KE is a recently developed app that provides matatu and buses stages information in the CBD

■ The application was developed as a final project in the University of Nairobi.

■ Users can be able to get the:

◘ Stops Made upto Destination

◘ Fares Charged, during Peak&OffPeak Hours

◘ Description of the Sacco's Physical Location

■ The app also provides a tool for getting a walking or driving route from an input start location

■ The application has an additional screen (OpenStreetMap) which provides other OSM location data

■ The application can be downloaded from Google Playstore by searching for 'MAT za CBD'

■ Alternatively, download the apk version using the button below
